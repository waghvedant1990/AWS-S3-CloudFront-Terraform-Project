# AWS-S3-CloudFront-Terraform-Project
# 🚀 AWS S3 Static Website Hosting using Terraform

## 📌 Project Overview

This project demonstrates how to deploy a production-grade static website on AWS using Terraform.

The infrastructure includes:

* AWS S3
* CloudFront CDN
* Route 53
* ACM SSL Certificate
* Terraform IaC

---

## 🌐 Live Website

https://devk.co.in

---

## 🏗 Architecture

User → Route53 → CloudFront → S3 Bucket
